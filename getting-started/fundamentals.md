1. What is an Operating System (OS)?
	An Operating System (OS) is software that acts as a bridge between hardware and user/applications.
	Examples:
	Linux
	Windows
	macOS

Main Responsibilities of OS:
	Hardware Management
	Memory Management
	Process Management
	Device Management
	File System Handling
	Security & User Management
	Error Handling & Logging

2. Why Linux Over Windows?
	Cost-Effectiveness
	Free and open-source (no license cost)
	Lower maintenance cost

Performance & Efficiency
	Uses fewer resources (lightweight)
	Works well even on low-end systems
	Highly scalable (from small devices → data centers)

Security & Reliability
	Strong user permission system
	Less vulnerable to viruses
	Frequent updates without reboot


High uptime (can run for years)

Interview Tip:
“Linux is preferred in servers due to stability, security, and cost advantage.”

Very Brief History of Linux

Created by Linus Torvalds in 1991
Inspired by UNIX
Started as a personal project
Now used in: Servers, Cloud, Android phones, Networking devices

Interview Line:
“Linux is an open-source Unix-like OS kernel developed in 1991.”
